12-month rule (delivery)
Fuld rapport frigives først når 12 betalte måneder er nået (baseret på betalte Stripe invoices) → sendes via email og låses op i dashboard.
What we automate over time
Audit generation
SEO/Ads findings, priorities, report artifacts.
Recommendations
Action list with effort/impact, 90-day plan.
Execution (hybrid)

Automation betyder ikke “uovervåget”. Vi starter med human-in-the-loop og bygger sikkerhed + logging før autopilot.

Contract
Preview vs Full
Preview (always visible)
Top issues + 90‑day plan + progress tracker. Ingen fulde keyword/content planer.
Full report (locked)
Fuld audit + konkrete ændringer + prioriteret backlog. Leveres efter 12 betalte måneder.
Dette skaber commitment og sikrer kvalitet over tid.
Gated delivery

System architecture

Bygget til reliability: web app hurtig, audits i baggrunden, artifacts gated.

Frontend
Vite + React
Landing, checkout redirect, dashboard, preview report UI.
API
Express + Prisma
Auth, sites, subscriptions, audit jobs, artifact access control.
Worker
Audit engine
Queue → run analysis → write preview/full artifacts → signal completion.
Artifact gating
Worker kan generere både preview og fuld rapport tidligt, men API’et beskytter fuld artifact indtil 12 betalte måneder. Herefter sendes email via Resend.

Build timeline

12 uger til en troværdig v1: subscription + preview report + gating + delivery. Derefter hybrid automation.
